Sock Problem

Problem

Link to problem on HackerRank.

There are n colors of socks in a sock drawer. You want to find a pair of socks that are the same color. How many socks must you take out to be sure of having a pair of the same color?

Solution:

I discovered that you simply needed to pull out one more sock than the number of matching pairs that you owned in the drawer.

Result: best case you get two pairs right of the bat. but if not then the maximum amount of pulls it would take would be one more than the amount of matching pairs you owned

Picture Diagram

Psuedocode

function howmanytopullout(n) return n + 1; n+1 

    Let n be how many sock colors  owned
    Result: Best case scenario is you pull two matching pair with your first two pulls. Worst case is you have to pull one more than the number of matching pairs you own.
